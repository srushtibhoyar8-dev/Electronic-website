#Electronics Shop Website

##Project Overview
This is a responsive Electronics Shop website built using HTML, CSS, and JavaScript.  
Users can browse products, search/filter items, add them to a cart, view total price, and submit a validated contact form.

---

##Features

✅ Responsive navigation bar with sidebar menu  
✅ Product listing loaded from JavaScript data  
✅ Real-time search functionality  
✅ Category filter option  
✅ Add to cart with localStorage support  
✅ Cart item count in navbar  
✅ Total price calculation  
✅ Contact form with strict validation  
✅ Fully responsive design (mobile, tablet, desktop)

---

##Technologies Used

- HTML5  
- CSS3 (Flexbox/Grid, Media Queries)  
- JavaScript (DOM, LocalStorage)

##Design Decisions

- Simple UI for better usability  
- Responsive layout using CSS Grid/Flexbox  
- LocalStorage used for cart persistence  
- Sidebar added for mobile navigation
